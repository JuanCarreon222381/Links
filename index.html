Ideas Principales.

    • La web funciona mediante links, que nos llevan a otras paginas, otros sitios web, imagenes, a escribr un e-mail entre otras cosas
      Un link puede ser cualquier cosa por ejemplo un texto, una imagen, un video, un boton.

    • Apuntes de Clase.

    Los Estados

    Por defecto, los enlaces aparecen de la siguente manera en todos los navegadores

    • Un enlace no visitado esta subrayado y azul
      Un enlace visistado esta subrayado y morado
      Un enlace eactivo esta subrayado y rojo

    • Resumen.

    Los links o enlaces en HTML se crean con la etiqueta "a" (anchor). Esta etiqueta permite conectar diferentes páginas, archivos o recursos, dentro o fuera de un sitio web.
